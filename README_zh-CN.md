# 自用 1Panel 应用商店

**语言：** [English](README.md) | 简体中文

一个为 [1Panel](https://github.com/1Panel-dev/1Panel) 定制的应用商店，支持第三方应用程序自动更新。

## 📱 精选应用

- **NapCat** - 用于构建聊天机器人的 QQ 框架
- **OI-Bot** - 智能聊天机器人平台

## ✨ 功能

- 🔄 **自动更新** - 通过 Renovate 管理版本更新
- 📦 **易于安装** - 简化安装脚本
- 🤖 **CI/CD 集成** - GitHub Actions 工作流用于自动化

## 🚀 快速入门

### 一键安装

运行以下单个命令以下载并执行安装脚本：

```bash
bash <(curl -sL "https://raw.githubusercontent.com/talentestors/appstore-1panel/refs/heads/main/install_app.sh")
```
